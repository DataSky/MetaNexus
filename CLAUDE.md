# MetaNexus Project Guidelines

## What is this?
MetaNexus is the universal discovery, trust, and settlement layer for the agent economy — "Google for Agents."

## Development Principles
1. Protocol-agnostic: Support A2A, MCP, UCP, and custom protocols
2. Trust-first: Every feature should strengthen the trust fabric
3. Agent-native: Design for agent users, not human users
4. Open by default: Protocol layer is open, platform layer adds value
5. Test discipline: Inherit MetaD's comprehensive testing approach

## Code Style
- TypeScript with strict mode
- Vitest for testing
- ESM modules
- Functional core, imperative shell
